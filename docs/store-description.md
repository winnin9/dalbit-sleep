# 앱인토스 스토어 등록 정보 (PRD v1.1 기반)

## 앱 기본 정보
- **앱 이름 (appName)**: `dalbit-sleep` (영문 하이픈 허용)
- **표시 이름 (displayName)**: 달빛수면
- **한 줄 설명 (30자 이내)**
  매일 수면을 기록하고 달빛 조각으로 토스 포인트 받기!

## 앱 상세 설명 (스토어 노출용)

🌙 어젯밤 일어난 일을 별점으로 남겨보세요.
달빛수면은 잠들기 전과 후의 기분을 간단히 남기고, 건강한 수면 습관을 만들어가는 앱입니다.

🌟 **기록만 해도 쌓이는 달빛 조각**
- 푹 잔 날은 물론, 뒤척인 날에도 수면 시간과 기분을 기록하면 '달빛 조각'을 드립니다.
- 모은 달빛 조각이 보름달로 모이면 **즉시 토스 포인트로** 받을 수 있어요!

💡 **첫 달은 3일 만에 완성!**
- 수면 기록을 처음 시작하시는 분들을 위해, 첫 보름달은 단 5조각(최단 3일) 만에 완성됩니다. 
- 이후로는 15조각마다 잊지 않고 보름달 리워드를 챙겨가세요.

📊 **나만의 수면 통계**
- 매달 나의 수면 패턴이 어떤지 한눈에 볼 수 있습니다. 
- 날마다 변하는 달의 위상을 보며 뿌듯함을 느껴보세요!

**※ 달빛수면은 토스 회원 누구나 별도 가입 절차 없이 안전하고 빠르게 이용할 수 있습니다.**
